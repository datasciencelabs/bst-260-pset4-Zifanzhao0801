  1. Examine the population matrix you just created. Notice that 1) it is not tidy, 2) the column types are not what we want, and 3) the first row is a header. Convert population to a tidy dataset. Remove the state ID column and change the name of the column with state names to state_name. Add a column with state abbreviations called state. Make sure you assign the abbreviations for DC and PR correctly. Hint: Use the janitor package to make the first row the header.
library(tidyverse)
library(janitor)
state_map <- setNames(state.abb, state.name)
population <- population |>
  as.data.frame(stringsAsFactors = FALSE) |>
  row_to_names(row_number = 1) |>            # 1) first row becomes header
  as_tibble() |>
  select(-state) |>                           # 2) drop state FIPS column
  rename(state_name = NAME) |>                # 3) rename NAME -> state_name
  pivot_longer(                               # 4) tidy: wide -> long
    cols = starts_with("POP_"),
    names_to = "year",
    values_to = "population"
  ) |>
  mutate(
    year = as.integer(str_remove(year, "^POP_")),     # 5) remove "POP_" and parse year
    population = as.numeric(population),              # 6) numeric population
    state = recode(state_name, !!!state_map),         # 7) add 50-state abbreviations
    state = case_when(                                # 8) add DC & PR
      state_name == "District of Columbia" ~ "DC",
      state_name == "Puerto Rico" ~ "PR",
      TRUE ~ state
    )
  )
